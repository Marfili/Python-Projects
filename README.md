# Parta Ola
**"Parta ola"** is my first small game project as a Computer Science student, created to practice Python and basic programming concepts.
A console-based multiplayer game written in Python where players compete by managing beans and interacting with a shared pot through random game events.
The goal of the game is simple: be the last player with beans remaining.

**Description:**
"Parta ola" is a turn-based simulation game where multiple players participate in rounds.
Each player starts with a fixed number of beans and interacts with a central pot through randomly selected actions.
Players may need to:
-add beans to the pot
-take beans from the pot
-or trigger actions that affect all players
The game continues until only one player remains.

**Features:**
-multiplayer support
-bean management system
-random event system
-turn-based gameplay

-round system
-game state tracking
